Konfigurasi :

1. Extrack FIle Donasi.zip
2. Terdapat file database 'db_abm.sql' , import databases ini ke Databases 'db_abm' di local server.
3. Pada Folder /donasi/model/config/ , terdapat file connect.php
	* Ubah host dengan alamat ip server ayobantumasjid.com
	* serta ubah pass dan user sesuai pass dan user database
4. copy file connect.php yang sudah dikonfigurasi ini, dan replace file databases.php yang ada di folder
	/donasi/portal-adm/mode/config/
5. Pada folder /donasi/portal-adm/libs/ terdapat file path.php
	ganti nama web 'rhs.it-unhas.org' dengan 'ayobantumasjid.com'
6. Setelah selesai, Upload folder donasi ke server ayobantumasjid.com.
7. sebelum menjalankan/test websitenya, pastikan url: ayobantumasjid.com/donasi/ yang sebelumnya memakai wordpress telah dihapus.
